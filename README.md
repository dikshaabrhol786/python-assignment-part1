#  Student Grade Tracker

##  Project Overview

This project is a command-line based **Student Grade Tracker** built using core Python concepts. It processes raw student data, cleans and transforms it, performs marks analysis, and generates a class performance report.

---

## Objectives

* Clean and structure raw student data
* Perform marks analysis using loops and conditionals
* Generate class-level performance insights
* Practice string manipulation and user input handling

---

##  Features

### 1️⃣ Data Parsing & Cleaning

* Removes extra spaces and formats names into Title Case
* Converts roll numbers from string to integer
* Transforms marks from string format into a list of integers
* Validates student names (alphabet-only check)
* Displays structured student profile cards

---

### 2️⃣ Marks Analysis

* Assigns grades based on marks:

  * A+ (90–100), A (80–89), B (70–79), C (60–69), F (<60)
* Calculates:

  * Total marks
  * Average marks
  * Highest and lowest scoring subjects
* Allows dynamic subject entry using a loop
* Handles invalid inputs gracefully

---

### 3️⃣ Class Performance Summary

* Computes average marks for each student
* Assigns Pass/Fail status (≥60 = Pass)
* Displays formatted class report
* Identifies:

  * Total pass and fail count
  * Class topper
  * Overall class average

---

### 4️⃣ String Manipulation Utility

* Cleans and formats a paragraph
* Converts text to Title Case
* Counts occurrences of a word
* Replaces words with emojis
* Splits text into sentences and formats output

---

##  Tech Stack
